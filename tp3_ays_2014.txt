1 - El comando convert permite manipular y realizar conversiones sobre distintos
tipos de archivos. Utilice el comando convert y la estructura for para crear
un script llamado jpg2pdf.sh tal que 
   * convierta todos los archivos de tipo jpg en el directorio actual, a 
     archivos de tipo pdf.
   * el nombre del archivo resultante debe ser igual al nombre original con 
     el agregado .pdf al final. Ejemplo, si el archivo original se llama 
     foto.jpg, el archivo resultante deberá llamarse foto.jpg.pdf

2 - Ejecute las siguientes sentencias: 
    CADENA=cadenalarga ; echo $CADENA ${CADENA%%larga}
    CADENA=cadenalarga ; echo $CADENA ${CADENA##cadena}
